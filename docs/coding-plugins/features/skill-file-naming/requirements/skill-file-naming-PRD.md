# Skill File Naming PRD

## 需求总览

REQ-001 要求插件仓库内部 skill supporting files、测试文件和可搜索资源路径保持稳定命名，避免大小写、序号式命名和 snake_case 测试路径继续扩散。

## 追踪矩阵

| ID | 验收口径 |
| --- | --- |
| REQ-001 | 命名规则必须由自动测试覆盖，并接入 preflight。 |
