## 摘要

本 feature 规范 Coding Plugins 文档的职责边界：frontmatter 是关系和检索的机器源，正式正文负责需求、设计、计划和证据，`INDEX.md` 只作为生成式检索视图。README 保留人工摘要和轻量例外追踪，不再维护手写产物链路。
