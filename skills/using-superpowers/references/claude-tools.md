# Claude Code 工具映射

同 `using-coding-plugins/references/claude-tools.md`。本文件保留给原 `using-superpowers` 入口引用。

| 原意图 | Claude Code 等价方式 |
| --- | --- |
| Read/Grep/Glob | 直接使用 Claude Code 的 Read/Grep/Glob |
| Edit/MultiEdit/Write | 直接使用 Claude Code 的编辑工具 |
| Bash | 使用 Bash，遵守权限模式 |
| TodoWrite | 使用 TodoWrite |
| Task/subagent | 使用 Task 或 Claude Code 子代理能力 |
| Plugin reload | 使用 `/reload-plugins` |
| Plugin validation | 使用 `claude plugin validate <plugin-path> --strict` |
