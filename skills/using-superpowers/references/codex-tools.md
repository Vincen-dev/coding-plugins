# Codex 工具映射

同 `using-coding-plugins/references/codex-tools.md`。本文件保留给原 `using-superpowers` 入口引用。

| 原意图 | Codex 等价方式 |
| --- | --- |
| Read/Grep/Glob | `sed`, `rg`, `find`, `rg --files` |
| Edit/MultiEdit | `apply_patch` 或项目格式化工具 |
| Bash | `exec_command` / shell session |
| TodoWrite | `update_plan` 或明确的任务清单 |
| Task/subagent | 可用的多代理工具；没有时主代理分批执行 |
| Browser | 浏览器控制、Playwright 或截图验证 |
