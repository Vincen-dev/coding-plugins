# Coding Plugins 2.1.0

Coding Plugins is a workflow-only skill collection. Start with `using-coding-plugins`, select the smallest safe risk profile, and use `change-capsule` only when durable multi-turn state is valuable.

Before production changes, state a Verifiable Contract with Outcome, Boundary, and Verification, then establish test-first evidence. Follow systematic execution, keep the smallest honest artifact set, and run and read fresh verification before completion claims.

The extension has no executable runtime and writes no hidden project state. Preserve debugging, review, commit safety, and profile escalation rules from the selected skills.

Version 2.0.0 was the breaking migration that removed the executable runtime and introduced the workflow-only contract retained by later releases.
